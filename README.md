# OS Fundamentals Lab

This repository contains small, focused experiments and mini-projects
to understand core Operating System concepts through implementation.

## Topics Covered
- Processes and threads
- Scheduling concepts
- Synchronization primitives
- Memory management (upcoming)

## Why this repo?
Operating Systems concepts are best understood by building, not memorizing.
Each folder contains a minimal, self-contained example.

## Tech Stack
- Language: C / C++
- OS: Linux (POSIX)
- Tools: gcc, pthreads
